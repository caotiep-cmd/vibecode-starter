# VibeCode Starter

Practical AI coding and vibe coding starter kit for beginners/non-coders who want to build small real products.

Live site: https://caotiep-cmd.github.io/vibecode-starter/

## What is inside

- Homepage with positioning and roadmap
- Beginner guide: first AI API call
- Tool comparison: Cursor vs Lovable vs Bolt
- Project tutorial: build a landing page with AI
- Free AI Builder Prompt Pack

## Positioning

Not generic AI education. This site focuses on practical build workflows:

1. Turn ideas into clear specs
2. Build prototypes with AI tools
3. Debug AI-generated code
4. Learn API basics safely
5. Deploy and get real feedback

## Monetization path

- Free practical content
- Prompt/template pack
- Honest tool/course reviews
- Affiliate links only where relevant
- Later: paid starter kit or workshop
