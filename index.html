The path

Học bằng cách build 7 project nhỏ.

Day 1

Landing page

Dùng AI viết offer, copy, FAQ, HTML/CSS và publish bằng GitHub Pages.

Day 2

Prompt system

Biến ý tưởng mơ hồ thành spec, task list, acceptance criteria và test cases.

Day 3

API call đầu tiên

Lấy API key, gọi model, hiểu cost/token, tránh lộ key và lỗi phổ biến.

Day 4

Chatbot mini

Build bot hỏi đáp trên tài liệu nhỏ cho cửa hàng, khóa học hoặc portfolio.

Day 5

Automation

Google Sheet + AI để phân loại lead, viết email nháp, tóm tắt form.

Day 6–7

Mini SaaS demo

Ghép UI + database + API thành MVP có link demo để xin feedback thật.
