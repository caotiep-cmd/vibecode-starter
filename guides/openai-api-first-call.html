Gọi AI API đầu tiên trong 20 phút

Mục tiêu: hiểu API key là gì, chạy một request đơn giản, biết lỗi phổ biến và không vô tình lộ key.

Bạn sẽ build gì?

Một script nhỏ gửi câu hỏi tới AI model và nhận câu trả lời. Đây là nền tảng để sau này build chatbot, tool tóm tắt, automation hoặc mini SaaS.

1. API là gì, nói đơn giản?

API giống như một cái quầy order. App của bạn gửi yêu cầu: “hãy tóm tắt đoạn này”, “hãy viết email này”, “hãy phân loại lead này”. Nhà cung cấp AI trả lại kết quả để app dùng tiếp.

API key giống mật khẩu thanh toán. Ai có key có thể dùng quota/tài khoản của bạn, nên không được paste public lên GitHub, Facebook, hay frontend code.

2. Chuẩn bị

  • Node.js đã cài trên máy.
  • Một API key từ nhà cung cấp AI bạn chọn.
  • Một folder project trống.

Quan trọng: guide này dùng placeholder YOUR_API_KEY. Khi publish website/app thật, đừng để key trong HTML/JS chạy trên browser.

3. Tạo file chạy thử

Tạo file first-ai-call.js:

const API_KEY = process.env.AI_API_KEY;

async function main() {
  if (!API_KEY) {
    throw new Error('Missing AI_API_KEY environment variable');
  }

  const response = await fetch('https://api.openai.com/v1/responses', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      model: 'gpt-4.1-mini',
      input: 'Explain APIs to a beginner in 3 bullet points.'
    })
  });

  if (!response.ok) {
    const errorText = await response.text();
    throw new Error(`API error ${response.status}: ${errorText}`);
  }

  const data = await response.json();
  console.log(data.output_text);
}

main().catch(console.error);

4. Chạy script

Trên terminal:

# macOS/Linux
export AI_API_KEY="your_key_here"
node first-ai-call.js

# Windows PowerShell
$env:AI_API_KEY="your_key_here"
node first-ai-call.js

5. Lỗi thường gặp

401 Unauthorized

Key sai, key hết hạn, hoặc biến môi trường chưa được set đúng.

429 Rate limit

Gửi quá nhiều request hoặc tài khoản/quota chưa đủ. Đợi một lúc hoặc kiểm tra billing/limits.

Model not found

Tên model không đúng hoặc tài khoản chưa có quyền dùng model đó. Đổi sang model rẻ/phổ biến hơn.

Fetch is not defined

Node quá cũ. Cập nhật Node.js bản mới.

6. Prompt debug khi bị lỗi

Act as a patient senior developer.

I am trying to make my first AI API call.
Goal: [what I expected]
Error: [paste exact error]
Code: [paste code]
Environment: Windows PowerShell / Node version / provider

Tell me the 3 most likely causes, what to check first, and the smallest safe fix.
