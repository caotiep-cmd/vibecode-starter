Build landing page bằng AI và publish miễn phí

Đây là project đầu tiên nên làm: ít rủi ro, thấy kết quả nhanh, và học được workflow idea → copy → code → publish.

1. Viết offer trước, đừng viết code trước

Landing page không phải bài tập HTML. Nó là câu trả lời cho 3 câu hỏi:

  • Ai là người dùng?
  • Họ đang đau ở đâu?
  • Trang này giúp họ làm gì nhanh hơn/tốt hơn?
Ví dụ: “A free AI prompt kit for freelancers who want to improve Upwork proposals in 20 minutes.”

2. Prompt tạo cấu trúc landing page

Act as a conversion copywriter and product designer.

Product: [your product/tool]
Target user: [who]
Main pain: [pain]
Main outcome: [outcome]
Proof/status: [free beta/demo/template]

Create a one-page landing page structure with:
1. Hero headline and subheadline
2. Primary CTA
3. 3 benefit cards
4. How it works section
5. Example use cases
6. FAQ
7. Final CTA

Tone: practical, specific, no hype.

3. Prompt tạo HTML/CSS

Turn this landing page structure into a single-file HTML page.

Requirements:
- Responsive design
- Clean modern style
- Inline CSS only
- No external libraries
- Clear CTA buttons
- Good SEO title and meta description
- Keep the copy specific and beginner-friendly

After code, give me a checklist to review before publishing.

4. Checklist trước khi publish

  • Title/meta description rõ.
  • CTA không bị placeholder vô nghĩa.
  • Mobile layout đọc được.
  • Không có API key/password trong code.
  • Link nội bộ không broken.
  • FAQ trả lời objection thật.

5. Publish miễn phí

Cách đơn giản nhất: tạo GitHub repo → upload index.html → Settings → Pages → Deploy from branch → chọn branch chính và root folder.

Nếu bạn chưa quen GitHub, nhờ AI hướng dẫn từng bước theo screenshot/trạng thái hiện tại. Đừng paste token/password vào chat công khai.

6. Sau khi có link live

Đừng ngồi polish. Gửi link cho 5 người trong target audience và hỏi: “Bạn có hiểu trang này giúp gì trong 5 giây đầu không?”
